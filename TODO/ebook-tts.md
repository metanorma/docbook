# E5 — TTS with highlighting

**Tier:** 4 (nice-to-have, advanced)

## Description

Text-to-speech using the Web Speech API with word-by-word highlighting. The currently spoken word/phrase is highlighted in the reader, similar to Immersive Reader.

## Considerations

- Browser support varies (Safari, Chrome)
- Requires user gesture to activate
- Highlighting needs to track position in the mirror document
