# Supported Elements

The gem defines 157 element classes covering the most common DocBook 5
elements. This reference lists all supported elements organized by
category.

== Structural elements

[cols="1,1,3"]
|===
| XML Element | Ruby Class | Notes

| `<article>`
| `Article`
| Top-level document; flat section hierarchy

| `<book>`
| `Book`
| Top-level document; part/chapter structure

| `<chapter>`
| `Chapter`
| Book chapter container

| `<part>`
| `Part`
| Book part container

| `<section>`
| `Section`
| Generic recursive section

| `<sect1>` -- `<sect5>`
| `Sect1` -- `Sect5`
| Numbered section levels 1--5

| `<appendix>`
| `Appendix`
| Book appendix

| `<preface>`
| `Preface`
| Book preface

| `<set>`
| `Set`
| Collection of books

| `<reference>`
| `Reference`
| Container for refentry elements

| `<topic>`
| `Topic`
| Standalone topic fragment

| `<dedication>`
| `Dedication`
| Dedication page

| `<acknowledgements>`
| `Acknowledgements`
| Acknowledgements page

| `<colophon>`
| `Colophon`
| Production notes

| `<simplesect>`
| `Simplesect`
| Non-recursive titled section
|===
