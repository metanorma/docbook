== demo

Build a demo interactive reader from the bundled DocBook sample. This is the quickest way to see the reader in action without providing your own DocBook file.

=== Usage

[source,bash]
----
$ docbook demo [options]
----

=== Options

[cols="3,1,4"]
|===
| Option | Default | Description

| `--output`, `-o` PATH | `demo.html` | Output HTML file path
|===

=== Examples

==== Default output

[source,bash]
----
$ docbook demo
Built demo at /path/to/demo.html
----

==== Custom output path

[source,bash]
----
$ docbook demo -o /tmp/sample.html
Built demo at /tmp/sample.html
----

=== What's included

The demo uses the bundled *DocBook XSLT 2.0 Stylesheet Reference Guide* as sample content, demonstrating:

* Multi-chapter book structure with table of contents
* Code listings, admonitions, tables, and images
* Cross-references and index terms
* All 4 themes (Day, Sepia, Night, OLED)
* Full-text search and breadcrumb navigation

Open the output file in a browser to explore the reader.
