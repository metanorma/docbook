== to-html (deprecated)

[WARNING]
====
The `to-html` command is deprecated. Use link:convert-command[`convert`] instead.
====

The `to-html` command still works but prints a deprecation warning:

[source,bash]
----
$ docbook to-html book.xml -o output.html
[DEPRECATED] Use 'docbook convert' instead of 'docbook to-html'
Written to output.html
----

See link:convert-command[convert] for current documentation.
