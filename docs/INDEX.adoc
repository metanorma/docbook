= DocBook Documentation
:toc: macro
:toclevels: 2

[.lead]
DocBook is a Ruby gem for parsing, validating, and converting DocBook 5 XML documents into interactive HTML readers and structured JSON.

toc::[]

== Welcome

DocBook helps you:

* **Build** interactive HTML readers from DocBook XML -- self-contained files that work offline
* **Parse** DocBook 5 XML into Ruby object models using 170+ element classes
* **Export** ProseMirror-compatible JSON (DocbookMirror) for rich document editing
* **Resolve** XIncludes with text fragid extensions (`search=`, `line=`, `char=`)
* **Validate** and **format** DocBook XML from the command line
